Exercício 1
O que é o MKDocs e quais são suas principais vantagens?

O MKDocs é uma ferramenta de código aberto utilizada para criar sites de documentação estáticos a partir de arquivos escritos em Markdown. É muito utilizada para documentar projetos de software devido à sua simplicidade e facilidade de configuração.

Principais vantagens
Fácil de usar: basta escrever a documentação em Markdown.
Configuração simples: utiliza apenas um arquivo (mkdocs.yml) para configurar o projeto.
Visualização em tempo real: possui um servidor local que atualiza automaticamente as alterações.
Geração de site estático: permite publicar facilmente em serviços como GitHub Pages.
Temas modernos: suporta diversos temas, como o Material for MKDocs, que oferece uma interface moderna e responsiva.

Exercício 5
Como o Markdown facilita a criação de documentação?

O Markdown é uma linguagem de marcação simples que permite criar documentos formatados utilizando uma sintaxe fácil de aprender. Em vez de utilizar editores complexos, o desenvolvedor escreve apenas texto com pequenas marcações para gerar títulos, listas, tabelas, imagens e blocos de código.

Vantagens do Markdown
Sintaxe simples e fácil de aprender, permitindo escrever documentação rapidamente.
Leitura facilitada, pois os arquivos continuam organizados mesmo em formato de texto puro.
Compatibilidade com diversas ferramentas, como MKDocs, GitHub, GitLab e editores de texto.
Controle de versão eficiente, pois arquivos Markdown são textos simples e funcionam muito bem com Git.
Portabilidade, podendo ser convertido para HTML, PDF e outros formatos sem grandes dificuldades.
