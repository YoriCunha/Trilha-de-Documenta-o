# Instruções


1. Instalar o Node.js
Baixe o instalador no site oficial.
Execute o instalador e conclua a instalação.

2. Configurar as variáveis de ambiente
Verifique se o diretório do Node.js foi adicionado ao PATH.
Caso necessário, configure as variáveis manualmente.

3. Testar a instalação
Abra o terminal e execute:
node -v
npm -v
Se os números das versões forem exibidos, a instalação foi concluída com sucesso.
